# AppD20 🎲

![Kotlin](https://img.shields.io/badge/Kotlin-100%25-7F52FF?logo=kotlin&logoColor=white)
![Android](https://img.shields.io/badge/Android-Application-3DDC84?logo=android&logoColor=white)
![SharedPreferences](https://img.shields.io/badge/Storage-SharedPreferences-2196F3)

Aplicativo Android simples desenvolvido em **Kotlin** para uma atividade de mobile.  
O projeto possui **duas telas principais**: uma para cadastrar o nome de um RPG e outra para realizar a rolagem de um **D20**, simulando uma mecânica básica de jogo de mesa.

---

## Funcionalidades

- Cadastro do nome de um RPG
- Rolagem de um dado **D20**
- Persistência simples de dados com **SharedPreferences**
- Interface objetiva e de fácil uso
- Navegação entre duas telas

---

## Tecnologias utilizadas

- **Kotlin**
- **Android SDK**
- **SharedPreferences**
- **Material Design**

## Como executar o projeto

1. Clone este repositório:
   ```bash
   git clone https://github.com/JorgeALDev/AppD20.git
   ```
2. Abra o projeto no **Android Studio**
3. Execute em um emulador ou dispositivo físico

---

## Estrutura básica

- **Tela 1**: cadastro do nome do RPG
- **Tela 2**: rolagem do D20 e exibição do resultado
- **SharedPreferences**: armazenamento simples do nome informado

---

## Objetivo do projeto

Este aplicativo foi desenvolvido como uma atividade acadêmica da disciplina de **Mobile**, com foco em conceitos básicos de desenvolvimento Android, navegação entre telas e persistência local de dados.

---

## Observações

Projeto simples, com foco didático, utilizando apenas funcionalidades essenciais para demonstrar a construção de uma aplicação Android funcional.
